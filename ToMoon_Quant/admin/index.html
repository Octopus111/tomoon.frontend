Emergency Stop Standard Operating Procedure (§8.2)

Trigger Conditions
ConditionThresholdScope
Intraday DD > L3-$600Portfolio
Consecutive losses ≥ 55 tradesPer-strategy
System anomalyFeed gap > 30s, unresponsiveGlobal
Manual triggerOperator decisionGlobal
Immediate Actions
  1. Cancel all pending orders
  2. Flatten all open positions at market
  3. Block all new entry signals
  4. Log kill event with timestamp, trigger, portfolio state
  5. Notify operator via Slack + email
Recovery Process
  1. Root cause identified and documented
  2. Postmortem filed with deviation analysis
  3. Eddie sign-off to resume (manual approval)
  4. Resume at L1 degradation (50% size) for first session
  5. Return to L0 only after 2 clean sessions
